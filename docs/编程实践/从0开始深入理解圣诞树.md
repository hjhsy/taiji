## Python绘制
### ❤文本绘制

#### 预览
```
     *
    ***
   *****
  *******
 *********
***********
     |
     |

```

#### 思路
```
第一行n个空格1个雪花字符
第二行n-1个空格3个雪花字符
第三行n-2个空格5个雪花字符
以此类推,知道空格为0的时候停止
最后再画树干
```

#### 代码
::: details 点击查看代码
```python
# 定义初始打印空格数
n=10
# 定义初始打印雪花字符数
word=1
# 定义树干长度
treeLen=n//2
# 打印个小鳄鱼助助兴
print("""
                    __  __
       _ _        /_@)_@) \          /^^\ /^\ /^^\_
    _/oo \____/~''. . .  '~\       /'\''  ~ ''~~' -'\_
   / '.'. ~.~.~.       .'    ~ |     /'\~~..''''.'' ''  ~\_
  ('_'_'_'_'_'_'_'_  ' :   '     \_/' '.''  . '.   .''  '.  ~\_
  ~V~V~V~V  \   ~\  '' '~  '   '' ~   `   ~  ''   ~\_
    /\~/\~/\~/\~/|/  '   ''  _   ' ~ ''  '    ~  '' __  '  ..  \_
 <-- --- ---.---.--/'   ''   /'  '\_ '' ': ~ ;;''    ' /''; \ ;'''''' '' ~\ _
    \~ '. . : .:: ~. :.  /_'''_'' \_' :'''_ : _ ''/''_' '_ \:_ '''' #''..\/\/\/~/\~ ''~~~~~O
 ~~ \-~ `---~~~---- \(_)(_)(_)/ ~ ~~' ~\(_)(_)(_)\_~_~_~_~_~/˜¤¹
""")


# 打印树头,雪花符号数每次+2,空格每次-1
while n>=0:
    print(' '*n,'*'*word,sep='')
    word+=2
    n-=1
# 打印树干
# 树干位置在树中间,因为打印树头最后一行空格数为0,雪花符号数为
# word(奇数),所以中间位置是word//2+1,所以|前空格数为word//2
for i in range(treeLen):
    print(' '*(word//2),'|',sep='')
```

### turtle

我们上面用文本完成了简单圣诞树的绘制,接下来我们使用turtle绘制,turtle是python的内置库,属于入门级的图形绘制函数库,我们可以很容易使用它,如果你对turtle还不了解,不妨先看看这个[turtle入门](https://blog.csdn.net/sandalphon4869/article/details/99443949)

### 代码
::: details 展开代码
``` python
import turtle
# 设置屏幕大小
import random
screen = turtle.Screen()
screen.setup(800,600)
# 设置画笔大小
turtle.pensize(10)
# 隐藏笔头
turtle.hideturtle()
# 设置绘制速度
turtle.speed('fastest')
''':argument
下面函数是用来绘制圣诞树
x,y:第一个星星的坐标
rows:绘制的星星行数
tagramStep:星星的间距
tagramLength:星星的边长
tagramCount:初始绘制星星的个数
'''
def drawTree(x,y,rows,tagramStep=5,tagramLength=30,tagramCount=1):
    # 提前计算树干x坐标,因为之后x会变化
    tx=x
    # rows代要绘制几行星星
    for row in range(rows):
        # 拷贝一份x用来绘制星星
        rowX=x
        # 绘制星星
        for count in range(tagramCount):
            # 随机画笔颜色
            turtle.color(random.choice(['red','green','blue','yellow','orange']))
            turtle.penup()
            # 绘制的初始位置
            turtle.goto(rowX, y)
            turtle.pendown()
            # 使箭头朝右方便绘制
            turtle.seth(0)
            # 随机数,如果为1就画圆,反之画星星
            if random.choice([0,1]):
                turtle.circle(tagramLength//2)
            else:
                # 先向后半个长度再回来,使得星星横坐标中心点在rowX位置
                turtle.back(tagramLength//2)
                turtle.forward(tagramLength)
                # 接下来重复旋转前进
                for i in range(4):
                    turtle.right(144)  # 顺时针旋转144°
                    turtle.forward(tagramLength)
            # 绘制完一颗星星后,计算下一个星星的初始位置,因为还是当前行,所以y坐标不变
            rowX+=(tagramLength+tagramStep)
        # 绘制完一行后,计算第二行的初始星星坐标
        # 横纵坐标减去一个星星长度加间距
        x-=(tagramLength+tagramStep)
        y-=(tagramLength+tagramStep)
        # 星星个数增加2
        tagramCount+=2
    # 绘制树干
    turtle.penup()
    turtle.goto(tx,y)
    turtle.seth(-90)
    turtle.pendown()
    turtle.forward(rows*tagramLength)
    turtle.down()
    

drawTree(0,270,8,tagramLength=20,tagramStep=20)
turtle.mainloop()
```
